# ESP32 16-Channel Automatic Time Controlled Relays

# Requiments
- 16 Channel Relay
- ESP32 38Pins
- 2 Dupont wire
- 
# Install 
- Download the firmware and flash using 0x0 offset
- 
# Wifi Key
- Wifi Name: `ESP32_16CH_Smart_Switch`

- Password: `ESP32-admin`

# Activation
- Goto wifi settings and connect to your home wifi
